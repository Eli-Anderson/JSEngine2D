Class: Color

Color(r, g, b, a)

new Color(r, g, b, a)

Creates a Color object which holds RGBA values with a toString method for easy printing and setting to the Canvas' Context (which needs colors in the form "rgba(r,g,b,a)".
Parameters:
Name Type Description
r int [0,255] The red component of the color
g int [0,255] The green component of the color
b int [0,255] The blue component of the color
a double [0,1] The alpha component of the color
